亿西
类型: 小程序

1. 介绍

亿西是一款专为政府开发的小程序，旨在为企业提供政策查看、订阅和报名等功能。通过该小程序，企业用户可以轻松获取政府发布的最新政策资讯、报名参与相关活动，从而享受政府的补贴和补助。同时，政府可以通过平台便捷地发布政策、公告，确保信息及时传达给目标企业。

2. 技术栈

项目使用了以下技术栈，以确保小程序的流畅性和安全性：

  • uniapp：用于开发跨平台小程序，支持微信、支付宝等多个平台。
  • Vue 2：项目的核心框架，简洁、高效，易于维护。
  • Crypto.js：用于数据的加密处理，确保数据传输的安全性。
  • uni-ui：为项目提供了丰富的组件，助力快速开发和优化用户体验。
合e通
类型: pc

1. 介绍

合e通是为合肥市政府开发的一款企业自由交易平台，专为本地企业提供便捷、安全的交易环境。平台前端包含首页、需求集市、产品集市和活动专区，后端包含需求管理、产品管理、活动管理和数据统计等功能模块。整个系统采用DES算法进行数据加密，确保数据传输的安全性。登录方式集成了政务网的一键登录，为用户提供了更加便捷的访问体验。前台和后台均由个人独立开发完成，确保开发质量与功能集成。

2. 技术栈

项目采用了以下主要技术栈，以保证开发效率和系统的稳定性：

  • Vue 2：用于构建前端用户界面，简单易用，易于维护。
  • Vuex：状态管理工具，用于管理应用状态，提高开发效率。
  • Element-UI：用于构建响应式UI组件，提升界面美观性和交互性。
  • Crypto.js：用于加密和解密数据，确保数据传输的安全性。
